Pipes

1) Card Num Hide

This pipe is useful for hiding card number in template
For example : {{dummyCardNumber}} it can be hidden like : {{dummyCardNumber | cardNumHide }}

2) Greet

This pipe is useful for salutation based on time in template (Date in hours is input)
{{today|date:'H'|greet}}

3) Remove Underscore

This pipe is useful for removing underscores in template (snake case to normal)
{{removeUnderscore|removeUnderScore}}

4) Start Case

This pipe is useful for captalizing first letter in each word in template (it also removes underscore or snake case)
{{removeUnderscore|startCase}}
